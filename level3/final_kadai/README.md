# ポートフォリオサイト 実装報告

## 1. 実装したこだわりポイント

- **デザインを崩さないようにした**
  課題1と2で作ったデザイン（背景の色合いや、カードの角丸など）をそのまま残せるように意識した。新しく追加した「Works（実績）」や「モーダル（ポップアップ）」、そして今回のお問い合わせフォームも、全体の雰囲気に合うように調整した。
- **セキュリティ（エスケープ処理）**
  ただ動くだけでなく、サイトを攻撃されないように、文字やURLを出力するコードの直前にesc_htmlなどを細かく書き足した。

## 2. 苦労したエラーと解決方法

- **エラー内容**:
  実績（Works）の投稿を作ったのに、実際のページにうまく表示されなかったり、リンクを押すと「ページが見つかりません」となってしまった。
- **原因と解決方法**:
  WordPressに新しい投稿の仕組みを作った直後は、URLのルールがうまく更新されないのが原因であった。管理画面の「設定」＞「パーマリンク」を開いて「変更を保存」ボタンを押すことで、URLのルールがリセットされて、無事に正しく表示されるようになった。
